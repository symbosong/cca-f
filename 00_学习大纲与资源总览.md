## 四、环境准备

在开始学习之前，请确保以下环境已就绪：

### 4.1 必备环境

```bash
# 1. Python 3.7.1+
python --version

# 2. 安装 Anthropic SDK
pip install anthropic

# 3. 获取 API Key
# 访问 https://console.anthropic.com → API Keys → Create Key

# 4. 设置环境变量（推荐）
export ANTHROPIC_API_KEY="your-api-key-here"

# 5.（可选）克隆课程仓库以本地运行 Notebook
git clone https://github.com/anthropics/courses.git
```

### 4.2 验证安装

```python
import anthropic

client = anthropic.Anthropic()  # 自动读取环境变量 ANTHROPIC_API_KEY

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=100,
    messages=[{"role": "user", "content": "Hello, Claude!"}]
)
print(response.content[0].text)
# 如果看到 Claude 的回复，说明环境配置成功！
```

## 五、学习建议

### 按角色推荐

| 角色 | 基础路径（模块 1-7） | 进阶路径（模块 8-13） | 预计总时间 |
|------|---------|---------|:--------:|
| **架构师** | 全部 7 模块 | 全部 6 进阶模块 | 45-55h |
| **后端开发** | 模块 1-3 + 模块 7 | 模块 8 + 11 + 12 | 25-30h |
| **前端开发** | 模块 1-2 + 模块 3 | 模块 10（Streaming）+ 13 | 15-18h |
| **DevOps** | 模块 1-2 | 模块 9（CI/CD 重点）+ 12 | 15-18h |
| **产品经理** | 模块 1-2（文档版）| 模块 13（成本部分）| 8-12h |
| **测试工程师** | 模块 1-2（评估重点）| 模块 12 + 11（Batch） | 15-18h |

### 快速学习路线（4.5 小时速成版 — 基础）

```
1h  → 读本教程的 01-03 模块文档（建立全景认知）
1h  → 跑 GitHub courses 的 anthropic_api_fundamentals/ Notebook
1h  → 跑 GitHub courses 的 tool_use/ Notebook
1h  → 读官方 Agent SDK 文档
30m → 读官方 MCP 文档
```

### 进阶学习路线（15 小时深度版）

```
🔴 最高优先级（生产必用）
3h  → 09_Agent_SDK深度指南.md（Agent SDK 架构、工具权限、Agent Loop）

🟡 高优先级（深度进阶）
2h  → 11_Streaming与多模态深度.md（流式传输 + Vision + PDF + Citations）
2h  → 13_安全护栏与生产部署.md（生产系统安全基线）
2h  → 14_成本优化与模型选择策略.md（Extended Thinking + 智能路由）

🟢 补充学习
1h  → 克隆 Cookbook 仓库跑示例
```

### 如何使用本教程

1. **基础部分（01-07）**：从 `01_Claude入门.md` 开始，按顺序阅读，每个文件是自包含的完整学习单元
2. **进阶部分（09-14）**：完成基础后按优先级阅读，可根据角色选择性学习
3. **快速参考**：`08_进阶指南.md` 是进阶部分的浓缩版，适合快速查阅
4. **跑代码**：每个文件中的代码示例都可以直接复制到 Python 环境运行
5. **做练习**：每个文件末尾都有练习题，建议动手完成
6. **看视频补充**：文档中标注了对应视频编号，遇到不理解的部分可以回看视频
7. **查阅官方文档**：文档中标注了相关官方文档链接，深入理解时参考
