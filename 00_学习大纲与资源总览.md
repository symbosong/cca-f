## 四、环境准备

在开始学习之前，请确保以下环境已就绪：

### 4.1 必备环境

```bash
# 1. Python 3.7.1+
python --version

# 2. 安装 Anthropic SDK
pip install anthropic

# 3. 获取 API Key
# 访问 https://console.anthropic.com → API Keys → Create Key

# 4. 设置环境变量（推荐）
export ANTHROPIC_API_KEY="your-api-key-here"

# 5.（可选）克隆课程仓库以本地运行 Notebook
git clone https://github.com/anthropics/courses.git
```

### 4.2 验证安装

```python
import anthropic

client = anthropic.Anthropic()  # 自动读取环境变量 ANTHROPIC_API_KEY

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=100,
    messages=[{"role": "user", "content": "Hello, Claude!"}]
)
print(response.content[0].text)
# 如果看到 Claude 的回复，说明环境配置成功！
```

7. **做练习**：每个文件末尾都有练习题，建议动手完成
